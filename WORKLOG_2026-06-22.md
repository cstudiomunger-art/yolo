# WORKLOG 2026-06-22 · 签证模块对齐「YOLOHAPYY 签证检查器」交付包

把 app 简化版签证判定（旧 P1–P5）整体替换为交付包**已人工核验**的引擎：8 政策 + 201 国次 grant + 派生矩阵，算法 1:1 移植自 `engine.py`。客户端 Swift 离线判定 + Supabase 新表 CMS + 后台可编辑。

## 修了什么（核心缺陷）
- **计时纠正**：旧版 `entry_plus_Nd` 从入境日直加 → 新版 `next_day_0000`（入境次日 0 时）+ 24h `by_hour` 精确时刻。旧版**少算一天**会导致超期被拒。
- **择优而非短路**：旧版按 priority 短路 → 新版 `scorePolicy`（条件数→空间→天数余量，余量仅 tiebreak）。
- **四维体检单**：space / time / **进出口岸双向** / condition（续程票·第三国·随团），补齐旧版缺失的出境口岸、续程票、随团门槛、24h/邮轮/东盟团整类政策。
- **allowed_area 省/地级混合码矩阵**判可达性（旧版粗 region 判）。
- **新增**黄牌方案 A（换同省可达城市/收紧离境）+ 方案 B（办 L 签）、freshness 临期告警、also_eligible。
