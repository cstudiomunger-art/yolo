# WORKLOG · 2026-06-08

## 主题：全局悬浮音频播放器（可拖动音符 + 迷你控制栏 + 景点内播放列表）

### 一、需求
音频播放时，屏幕上常驻一个**可拖动的悬浮音符按钮**；点击展开**迷你控制栏**（上一首 / 下一首 / 进度条 / 播放列表）；音符右上角 **X** 同时关闭按钮与音频。播放列表范围＝**当前景点内**（主音频 + 各子区域音频）；悬浮按钮在**全 App** 范围常驻；切到未解锁音频时**自动进入试听**（沿用 180 秒试听 + 付费墙）。

### 二、架构改造
原先音频是**逐组件**的：每个 `AudioGuideSection`/`CityGuideAudioSection` 各持有一个 `@StateObject AudioPlaybackController`，无全局播放器、无队列、无上一首/下一首。本次升级为 **`AppEnvironment` 持有的单一全局队列播放器** + **`MainTabView` 顶层悬浮 UI**。

改动文件：
- `YOLO/Features/Guide/AudioPlayerView.swift`
  - `AudioPlaybackController` 演进为 `AudioQueuePlayer`（`@Observable`），保留 AVPlayer 播放核心（本地优先 / 远程降级、试听封顶 `effectiveMaxTime`、`AudioNowPlayingService` 锁屏控制）。
  - 新增队列：`queue`/`currentIndex`/`currentTrack`，`play(queue:startIndex:)`、`next()`、`previous()`、`playTrack(at:)`、`close()`；整曲播完自动续播下一首。
  - 新增 UI 状态：`isVisible`、`isExpanded`、`dragPosition`；权限回调 `resolveAccess`、试听结束回调 `onTrialEnded`、`refreshCurrentTrackAccess()`、`reloadIfCurrent(guideId:)`。
  - 新增 `AudioTrack` 模型（封装 guide + 标题 / 作者 + attraction/subArea 路由 + allowsPreview/isFree）。
- `YOLO/App/AppEnvironment.swift`
  - 持有 `let audioPlayer = AudioQueuePlayer()`；`bindAudioPlayer()` 绑定 `resolveAccess`（复用 `purchase.hasContentAccess` + `contentMode.effectiveUseRemoteIAP` + `branding.freeAudioPreviewSeconds`，镜像 `AudioGuideSection.hasFullAccess` 逻辑）；`signOutAndReset()` 调用 `audioPlayer.close()`。
- `YOLO/Features/Guide/MiniAudioPlayerView.swift`（新增，333 行）
  - 折叠态：可拖动音符按钮（子视图 `FloatingNoteButton`），右上角 X 关闭并停止音频。
  - 展开态：迷你控制栏（标题、上一首 / 播放暂停 / 下一首、进度条、时间、试听提示、播放列表入口）。
  - 播放列表面板：高亮当前曲、锁定曲显示 🔒、点选 `playTrack(at:)`。
  - 试听到时由 `onTrialEnded` 驱动 `.sheet(item:)` 弹 `MembershipPlansView`，解锁后 `refreshCurrentTrackAccess()`。

### 三、悬浮按钮拖动「不实时跟手」排查（三轮迭代）
症状：拖动时图标不动，只在松手后跳到最终位置。

1. **第一轮**：抽出 `FloatingNoteButton` 子视图，用 `@GestureState` 承载位移 → 仍不动。
2. **第二轮**：改为 `onChanged` 逐帧写回 `player.dragPosition` → 仍不动。
3. **坐标空间反馈环**：`DragGesture` 默认 `.local` 坐标系随 `.position` 移动而移动，`translation` 被自身位移抵消。改用 `coordinateSpace: .global` → 仍不动。
4. **真正根因**：音符是 **`Button`**，其内建点按手势**吞掉连续触摸流**，导致容器上的 `DragGesture` 拖动中收不到 `onChanged`，只在松手收到 `onEnded`。
   - **修复**：音符由 `Button` 改为普通 `Image` + `.onTapGesture`（点击展开）；拖动改用 `.highPriorityGesture`（稳获连续触摸流）；保留 `coordinateSpace: .global` + 逐帧写回 + 边界 `clamp`。`minimumDistance: 6` 区分点按与拖动。
